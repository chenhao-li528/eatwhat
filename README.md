# EatWhat 外卖点餐系统

## 项目简介
一个基于 Spring Boot + Vue3 的外卖点餐系统。

## 技术栈
- 后端：Spring Boot + MyBatis-Plus + JWT + MySQL
- 前端：Vue 3 + Element Plus + Axios + Pinia

## 功能模块
- 用户注册/登录
- 菜品浏览与分类筛选
- 购物车管理
- 订单管理

## 快速启动
### 后端
1. 创建数据库 eatwhat
2. 修改 application.yml 中的数据库配置
3. 运行 EatwhatApplication.java

### 前端
\`\`\`
cd eatwhat-frontend
npm install
npm run dev
\`\`\`
